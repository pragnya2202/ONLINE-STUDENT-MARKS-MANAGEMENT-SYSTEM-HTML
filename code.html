Online Student Marks Management System

📌 System Overview

This web-based application is used by schools to:

  • Store student details
  • Enter and manage marks
  • Calculate grades automatically
  • Generate performance reports

1️⃣ Admin Module

  • Add / Edit / Delete Teachers
  • Add / Edit / Delete Students
  • Manage Subjects
  • Assign Classes
  • View Overall Reports

2️⃣ Teacher Module

3️⃣ Student Module

Students can:

  • Login securely
  • View Marks
  • View Grades
  • Download Report Card

📌 System Architecture (3-Tier Structure)

🔹 Presentation Layer

HTML, CSS, JavaScript (Forms, UI)

🔹 Application Layer

Business Logic, Grade Calculation, Authentication (PHP / Python / Java / Node.js)

🔹 Database Layer

Data Storage (MySQL / PostgreSQL)

📌 Database Structure

Students Table
  • Student_ID (Primary Key)
  • Name
  • Class
  • Section
  • Roll_No
Teachers Table
  • Teacher_ID (Primary Key)
  • Name
  • Subject
  • Email
Subjects Table
  • Subject_ID
  • Subject_Name
  • Class
Marks Table
  • Mark_ID
  • Student_ID (Foreign Key)
  • Subject_ID (Foreign Key)
  • Marks
  • Grade

📌 Basic Flow of the System

  1. User logs in
  2. Teacher enters marks
  3. System calculates total & grade
  4. Data saved in database
  5. Student views result
